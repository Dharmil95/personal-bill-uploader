# Bill Uploader

A mobile-first application for capturing, categorizing, and managing bills.

## Features

- Camera capture and file selection for images and PDFs
- Category selection and custom categories
- Upload progress feedback
- Recent upload history with category filters
- Mobile-first responsive interface

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Status

This is currently a demo. Uploads are simulated in the browser and recent items are cleared when the page is refreshed.
